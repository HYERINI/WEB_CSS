<head>
    <meta charset="UTF-8">
    <title>첫번째 CSS</title>
    주로 이 자리에 간단한 css 코드를 적는다
</head>
==================================================================
*{
    margin: 0;
}

-> 전체 요소에 해당 css를 적용

==================================================================
p {
    font-style: italic;
}

-> html파일 안에 있는 모든 p 요소에 css를 적용

==================================================================
.classname {
    font-style: italic;
}

<h1 class="classname"></h1>
<span class="classname"></span>

->해당 클래스의 값을 갖고 있는 요소에 css를 적용

==================================================================
#idname {
    width: 500px;
    margin: 10px auto;
}

<div id="idname"></div>

->해당 id 값을 갖고 있는 요소에 css를 적용

=================================================================
h1, p {
    font-style: italic;
}

->css 내용이 같으면 하나로 합칠 수 있음
